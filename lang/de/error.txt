====== Backup Tool ======

**Backup Tool ist mir Ihrer aktuellen Konfiguration nicht funktionsfähig.**

Es gibt zwei Wege dieses Problem zu lösen:
  * Installieren Sie [[http://pear.php.net/package/PEAR|PEAR]], eine PHP-Bibliothek die eine Archivierungsklasse enthält, die von Backup Tool genutzt werden kann.
  * Erlauben Sie den Zugriff auf die PHP [[http://us.php.net/exec|exec]] Funktion und zusätzlichen Zugriff auf die Kommandozeilen basierten Tools **tar** und entweder **gzip** oder **bzip2**.

Wenn eins der beiden Probleme auf Ihrem System gelöst ist, aktualisieren Sie die Backup Tool Seite. Diese Fehlermeldung wird dann verschwinden und Sie können eine Sicherheitskopie anstoßen.
