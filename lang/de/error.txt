====== Backup Tool ======

**Backup Tool ist mir Ihrer aktuellen Konfiguration nicht funktionsfähig.**

Es gibt zwei Wege dieses Problem zu lösen:
  * Installieren Sie [[http://pear.php.net/package/PEAR|PEAR]], eine PHP-Bibliothek die eine Archivierungsklasse enthält, die von Backup Tool genutzt werden kann.
  * Erlauben Sie den Zugriff auf die PHP [[http://us.php.net/exec|exec]] Funktion und zusätzlichen Zugriff auf die Kommandozeilen basierten Tools [[wp>tar_(file format)|tar]] und entweder [[wp>gzip]] oder [[wp>bzip2]].

Sollten Sie eines der oben genannten Probleme im Bezug auf PHP gelöst haben, aktualisieren Sie bitte die Backup Tool Seite und leeren Sie den Cache. Diese Fehlermeldung wird dann verschwinden und Sie können erneut eine Sicherung anstoßen.
