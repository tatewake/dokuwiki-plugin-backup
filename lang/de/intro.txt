====== Backup Tool ======

Dieses Tool erstellt zuverlässig Sicherungskopieren von Ihrem Wiki für
den Fall, dass ein [[wp>Meteorit]] Ihren Hoster trifft oder ähnliche
Katastrophen eintreffen. Mithilfe der unten stehenden Checkboxen kann
eine beliebige Kombination an Inhalten gesichert werden.

Die erzeugte [[wp>tar_(file format)|tar]]-Datei, die
entweder mit [[wp>bzip2]] oder [[wp>gzip]] komprimiert wird, kann
daraufhin heruntergeladen und ggf. wieder gelöscht werden. Wenn keine
Komprimierungssoftware zur Verfügung steht, wird einfach die tar-Datei
unkomprimiert erzeugt.

**Hinweis:**

  * Wenn das Erstellen der Sicherungskopie einen Timeout erzeugt, versuchen Sie weniger Auswahl in eine Sicherungskopie zu packen.
  * Die erstellte Datei erhält einen Namen nach dem Format ""//dw-backup-JahrMonatTag-StundeMinuteSekunde.tar.bz2//"", mit der Zeit im 24-Stundenformat.
    * Die Datei ist in dieser Form formatiert, damit Sicherungskopien in chronologischer Folge sortiert werden.
  * Die Sicherung enthält **nicht** das komplette DokuWiki. Zur Wiederherstellun muss die gewünschte Version von [[http://download.dokuwiki.org/|DokuWiki.org]] heruntergeladen werden. Das Backup kann anschließend problemlos via Copy&Paste in das Wiki integriert werden.
  * **Wenn Sie sicherheitsrelevante Daten** (z.B. private Seiten oder Konfigurationsdateien in denen Passwörter enthalten sind) abspeichern, ist es empfehlenswert, den backup Namensraum so zu konfigurieren, dass nur Administratoren ihn einsehen können oder Sie löschen die Sicherungskopien sofort nachdem sie heruntergeladen wurden.
  * Sie können spezifische Verzeichnisse von der Sicherungskopie ausschließen (z.B. alte Sicherungskopien oder Cacheverzeichnisse von [[doku>plugin:latex|LateX plugin]]). Schauen Sie dazu in die Konfigurationsdatei dieses Plugins.
    * Die Sicherungsmethode "//GNU Tar (fast;unfiltered)//" ignoriert normalerweise diese Filter.
  * Sollte eine der Sicherungsmethoden ("//PEAR Archive Library//", "//GNU Tar (filtered)//" oder "//GNU Tar (fast;unfiltered)//") nicht funktionieren, versuchen Sie es bitte erneut mit einer anderen Sicherungsmethode.
